SOCKS5TunnelEventContains data and information from various events raised within a SOCKS5Tunnel instance.
Constants
ONCONNECTDispatched when the SOCKS5Tunnel instance successfully connects to the tunnel socket.
ONCONNECTERRORDispatched when the SOCKS5Tunnel instance experiences an error in connecting to the tunnel socket.
ONAUTHENTICATEDispatched when the SOCKS5Tunnel instance successfully authenticates the tunnel socket.
ONDISCONNECTDispatched when the SOCKS5Tunnel instance disconnects from the tunnel socket.
ONHTTPRESPONSEDispatched when the SOCKS5Tunnel instance received a complete response to a tunneled HTTP request.
ONHTTPREDIRECTDispatched when the SOCKS5Tunnel instance receives a 301 or 302 (redirect) HTTP response and the URLRequest used to initiate the request includes the followRedirects property as true.
ONHTTPMAXREDIRECTSDispatched when the SOCKS5Tunnel instance has done more than the maximum number of automated redirects (default is 5).
ONHTTPHEADERSDispatched when the SOCKS5Tunnel instance receives enough information to parse HTTP headers from a response.
ONHTTPSTATUSDispatched when the SOCKS5Tunnel instance receives enough information to parse HTTP status information from a response.

public static const ONCONNECT: String

Dispatched when the SOCKS5Tunnel instance successfully connects to the tunnel socket.

ONCONNECTERROR

public static const ONCONNECTERROR: String

Dispatched when the SOCKS5Tunnel instance experiences an error in connecting to the tunnel socket.

ONAUTHENTICATE

public static const ONAUTHENTICATE: String

Dispatched when the SOCKS5Tunnel instance successfully authenticates the tunnel socket.  Tunnel requests are available after this event is dispatched.

ONDISCONNECT

public static const ONDISCONNECT: String

Dispatched when the SOCKS5Tunnel instance disconnects from the tunnel socket.

ONHTTPRESPONSE

public static const ONHTTPRESPONSE: String

Dispatched when the SOCKS5Tunnel instance received a complete response to a tunneled HTTP request.  In a typical response, status would be parsed first, followed by headers, and finally by this event.

ONHTTPREDIRECT

public static const ONHTTPREDIRECT: String

Dispatched when the SOCKS5Tunnel instance receives a 301 or 302 (redirect) HTTP response and the URLRequest used to initiate the request includes the followRedirects property as true.

ONHTTPMAXREDIRECTS

public static const ONHTTPMAXREDIRECTS: String

Dispatched when the SOCKS5Tunnel instance has done more than the maximum number of automated redirects (default is 5).

ONHTTPHEADERS

public static const ONHTTPHEADERS: String

Dispatched when the SOCKS5Tunnel instance receives enough information to parse HTTP headers from a response.  In a typical response, status would be parsed first, followed by headers, and finally by the ONHTTPRESPONSE event.

ONHTTPSTATUS

public static const ONHTTPSTATUS: String

Dispatched when the SOCKS5Tunnel instance receives enough information to parse HTTP status information from a response.  In a typical response, headers would be parsed next, followed by the ONHTTPRESPONSE event.
